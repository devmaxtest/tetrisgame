# Feature Roadmap for Tetris Max

Cette roadmap liste les fonctionnalités et améliorations prévues pour les prochaines versions de Tetris Max.

## 🌟 Priorités actuelles :

### 1. Core Gameplay Features
- [x] Implémentation de la mécanique des pièces (déplacement, descente rapide, rotations).
- [x] Élaboration du calcul des collisions et placement des pièces.
- [x] Ajout de la logique pour comptabiliser les scores en fonction des lignes complétées.

### 2. Animations et UI 
- [x] Fluidité des animations des mouvements (gauche/droite, chute).
- [x] Intégration d'un effet visuel pour le drop instantané.
- [x] Transition d'état à l'écran (Début, Pause, Game Over).

## 🚀 Objectifs à moyen terme :

### 1. Optimisation 
- [ ] Réduction des temps de chargement initiaux.
- [ ] Limiter l'impact CPU et mémoire dans des parties longues.
- [ ] Revue des performances du moteur de logique.

### 2. Interface Utilisateur
- [ ] Améliorer la réactivité pour les écrans de petite taille.
- [ ] Personnalisation des contrôles clavier depuis les paramètres.

## 🔮 Long terme (expérimental) :

### 1. Fonctionnalités avancées
- [ ] Mode multijoueur : 1v1 avec compétition sur des grilles synchronisées.
- [ ] Enregistrement des parties et du scoring.

### 2. Community & Déploiement
- [ ] Leaderboard global en ligne.
- [ ] Hébergement via Hetzner, avec diagnostic serveur (logs et stabilité).
